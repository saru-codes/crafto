# Crafto Overview

Crafto is a multi-tenant website builder platform built with Laravel.

## MVP Goals
- Create unlimited websites ("sites") from one dashboard
- Build pages using reusable sections (section-based builder)
- Blog module per site
- Listings module with fast search + filters
- Ecommerce module with Stripe checkout
- SEO basics (slugs, meta title/description, sitemap later)

## Non-goals (MVP)
- Full Webflow-style drag/drop designer
- Advanced ecommerce features (coupons, shipping zones, tax engine)
