# Crafto Roadmap

## Phase 1 — Foundation
- Laravel project setup
- Auth + roles
- Multi-site model (Site, Site Domains, Settings)
- Filament admin setup
Deliverable: create/manage Sites from dashboard

## Phase 2 — Page Builder MVP
- Pages + Page Sections (JSON props)
- Section library (hero, features, CTA, FAQ, gallery, contact)
- Preview + publish
Deliverable: build and publish marketing pages

## Phase 3 — Blog
- Posts, categories, tags
- Blog list + post detail pages
- SEO fields
Deliverable: full blog per site

## Phase 4 — Listings + Search/Filters
- Listing types + dynamic attributes
- Listing CRUD + media
- Meilisearch indexing
- Faceted filters + sorting
Deliverable: directory website per site

## Phase 5 — Ecommerce MVP
- Products (+ variants optional)
- Cart + Stripe Checkout
- Orders dashboard
Deliverable: basic ecommerce per site

## Phase 6 — SaaS Expansion
- Cashier subscriptions + plans
- Plan limits
- Teams/roles per tenant
- Domain mapping + SSL automation
- Templates system (starter templates)
